## Project Description

A Customer Relationship Management (CRM) application built using React, Node.js, Express, and MongoDB Atlas.

## Features

* User Registration
* User Login
* JWT Authentication
* Protected Dashboard
* Lead Management
* MongoDB Atlas Integration
* Responsive UI

## Tech Stack

### Frontend

* React.js
* React Router

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Deployment

* Vercel (Frontend)
* Render (Backend)
